# nifty-ai

A personal trading assistant for the Indian markets that combines live Nifty 50, Bank Nifty, and India VIX data from the Kite Connect API with an Anthropic-powered chat interface. It builds market context from price action, technical indicators, and tracked positions, then lets you converse with an AI about setups, risk, and trade ideas — with a lightweight web frontend and a local trade log for reviewing past activity.
